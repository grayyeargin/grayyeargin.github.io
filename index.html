Brewfinder

Full Stack Web Developer

In producing Brewfinder I created a RESTful backend utilizing Ruby on Rails and PostgreSQL to serve the client-side customized JSON responses. The database was populated using Nokogiri and custom rake taks to scrape the web for “beer data”.

The client-side is built using the Backbone.js framework to create a one page application, rendering Underscore.js templates for each route. D3.js is used to create statistical data visualization, as well as creating a map for Brewery searches.

Technology Used

  • Ruby on Rails
  • Backbone.js
  • PostgreSQL
  • D3.js
  • JSON
  • CSS3
  • jQuery
  • Underscore.js
  • Adobe Photoshop

MusicMap

Full Stack Web Developer

MusicMap is a web application designed to aggregate information on musical artist's to give the user more information in one search. To do this my team and I consumed multiple 3rd party API's and directed them to one page for each queried artist. This query would produce a page with relevant Twitter, Spotify, lastFM, and Reddit information on the artist. Bootstrap is utilized for page styling with jQuery effects on the album transitions on the home page.

About Gray

I focus on enhancing the user experience by creating quick, responsive, cross-browser compatable web applications that utilize the industries latest web technology and best practices.
